Image Search Abstraction Layer

User Stories:

1) I can get the image URLs, alt text and page urls for a set of images relating to a given search string.

2) I can paginate through the responses by adding a ?offset=2 parameter to the URL.

3) I can get a list of the most recently submitted search strings.

Example output:

{ "url": "http://blog4free.com/wp-content/uploads/2017/05/funny-cats-pictures-when-it-went-nobody-knows-but-its-gone.jpg", "snippet": "Funny Cats Pictures Lolcats: I let my mind wander. – Blog4Free.com", "thumbnail": "https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcT-RtX7YyCOpFQyrkTPbvJBeBIHjS1S8KjfCPstVcIiotW9GoAEJ0GnAPQ", "context": "http://blog4free.com/images/funny-cats-pictures-lolcats-i-let-my-mind-wander/" }

Usage:

https://img-search-jh.glitch.me/api/lolcats%20funny?offset=10
